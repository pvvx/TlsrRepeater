# TlsrRepeater
Zigbee repeater (router) based on TLSR825x chipsets

Альтернативная прошивка для [Zigbee Repeater HOBEIAN:CK-BL702-ROUTER-01(7018)](https://pvvx.github.io/ZG-807Z/)
<img width="402" height="791" alt="image" src="https://github.com/user-attachments/assets/f147f25e-4763-4257-b695-926589e5e717" />

Это простой Zigbee роутер для автономного (резервного) питания от Li-ion АКБ.
Добавлен прямой “binding” кнопки (On/Off).
