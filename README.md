# TlsrRepeater
Zigbee repeater (router) based on TLSR825x chipsets

Альтернаивная прошивка для [Zigbee Repeater HOBEIAN:CK-BL702-ROUTER-01(7018)](https://pvvx.github.io/ZG-807Z/)

Это простой Zigbee роутер для автономного (резервного) питания от Li-ion АКБ.
Добавлен прямой “binding” кнопки (On/Off).
